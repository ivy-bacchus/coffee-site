# Coffee Study Site — 要件定義書

## 概要

コーヒーの基礎知識がある層を対象に、産地・農園・精製法・抽出の科学などを深く学べるウェブサイト。
洗練されつつも親しみやすいデザインで、読み物として完成されたコンテンツを提供する。

---

## 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| フレームワーク | Astro | 静的サイト生成に最適、高速、GitHub Pages対応 |
| UIコンポーネント | Astro Islands + React | インタラクティブ要素（レシピ計算機、産地マップ）のみReact |
| 地図ライブラリ | react-simple-maps | SVGベースの軽量な世界地図表示 |
| スタイリング | Tailwind CSS | 一貫したデザイントークン管理、高速開発 |
| デプロイ先 | GitHub Pages（当面） | 将来的にVercel移行も視野 |
| 画像 | AI生成 | 統一感のあるビジュアル |
| i18n | 自前実装（JSON辞書） | 日本語 / 英語の2言語切り替え |

## デザイン方針

### トンマナ

- **カラーパレット**: ダークブラウン / クリーム / アクセントにテラコッタ系
  - Primary: `#3C2415`（深煎りブラウン）
  - Secondary: `#D4A574`（キャラメル）
  - Background: `#FDF6EE`（クリーム）
  - Accent: `#C1553B`（テラコッタ）
  - Text: `#2C1810`（ダークブラウン）
- **雰囲気**: カフェのメニュー風 — 手に取れる質感、温かみのある紙のテクスチャ感
  - 罫線やディバイダーにメニューボード風の装飾
  - セクションタイトルに黒板風・手書き風のあしらい
  - カード要素には軽いシャドウと角丸で「メニューカード」感
- **フォント**:
  - 見出し: セリフ体（Playfair Display / Noto Serif JP）— メニュータイトル風
  - 本文: サンセリフ体（Inter / Noto Sans JP）— 読みやすさ重視
  - アクセント: 手書き風フォント（Caveat）— 注釈やラベルに使用
- **画像スタイル**: 温かみのある自然光風、マクロ撮影風のAI生成画像

### レイアウト

- レスポンシブ対応（モバイルファースト）
- **サイドバーナビゲーション**（左側固定）
  - セクション一覧を常時表示
  - 現在地のハイライト
  - モバイルではハンバーガーメニューに折りたたみ
- メインコンテンツ領域: サイドバー右側、最大幅 `960px`
- 記事ページは読みやすさ重視の最大幅 `720px` 程度のシングルカラム
- **トップページ**: ダッシュボード型レイアウト
  - セクションごとのカードをグリッド配置
  - 各カードにサムネイル画像 + タイトル + 短い説明文
  - カフェメニューの「本日のおすすめ」的な見せ方

```
/

│
├── /origins — 産地・農園
│   ├── 概要（インタラクティブ世界地図 — 国をクリックで詳細ページへ遷移）
│   ├── /origins/ethiopia — エチオピア
│   ├── /origins/kenya — ケニア
│   ├── /origins/colombia — コロンビア
│   ├── /origins/brazil — ブラジル
│   ├── /origins/guatemala — グアテマラ
│   ├── /origins/panama — パナマ（ゲイシャ種の深掘り）
│   ├── /origins/yemen — イエメン
│   └── ... （順次拡充）
│       各ページ: 地理・気候、代表的農園、栽培品種、精製法、フレーバープロファイル
│
├── /processing — 精製法
│   ├── ウォッシュド
│   ├── ナチュラル
│   ├── ハニー / パルプドナチュラル
│   ├── アナエロビック（嫌気性発酵）
│   └── 精製法とフレーバーの関係
│
├── /science — コーヒーの化学
│   ├── 焙煎の化学（メイラード反応、カラメル化、ストレッカー分解）
│   ├── 抽出の科学（溶解度、TDS、収率）
│   ├── 水質と抽出（硬度、pH、ミネラル組成）
│   └── 鮮度と劣化（酸化、脱ガス、保存方法）
│   ※ 図解は要所のみ（焙煎カーブ、抽出収率チャート程度）。テキスト主体で深く解説

├── /brewing — 抽出ガイド
│   ├── 抽出理論の基礎（変数の整理: 湯温、粒度、時間、比率）
│   ├── ドリップ（V60、カリタウェーブ、メリタ等）
│   ├── 浸漬式（フレンチプレス、クレバー）
│   ├── エスプレッソ
│   └── その他（サイフォン、エアロプレス、水出し）
│
├── /tools — レシピ計算機（インタラクティブ）
│   └── ドリッパー / 器具選択 → 杯数設定 → レシピ出力
│
├── /cupping — カッピング・テイスティング
│   ├── SCAカッピングプロトコル
│   ├── フレーバーホイール解説
│   └── テイスティングノートの書き方
│
└── /varieties — 品種図鑑
    ├── アラビカ種の系譜（ティピカ、ブルボン、ゲイシャ...）
    ├── ロブスタ種
    └── 品種とフレーバーの関係
```

## インタラクティブ要素

### 産地マップ（React Island）

- SVGベースの世界地図コンポーネント
- コーヒーベルト（北緯25度〜南緯25度）をハイライト表示
- 生産国をクリック / タップで該当の産地詳細ページへ遷移
- ホバー時にツールチップ（国名、代表品種、フレーバーキーワード）
- ライブラリ候補: `react-simple-maps` or カスタムSVG

---

### レシピ計算機（React Island）

### 入力項目

1. **抽出器具の選択**
   - ドリッパー: V60（円錐）、カリタウェーブ（フラットボトム）、メリタ（台形1つ穴）
   - 浸漬式: フレンチプレス、クレバードリッパー
   - その他: エアロプレス、サイフォン
2. **仕上がり杯数**（1〜5杯、1杯 = 約150ml）
3. **好みの濃さ**（ライト / ミディアム / ストロング）

### 出力項目

- 豆の量（g）
- 湯量（ml）
- 推奨湯温（℃）
- 推奨挽き目（粗さの目安）
- 注湯手順（ステップバイステップ）
- 目安の抽出時間

## 多言語対応（i18n）

- ヘッダーに **JP / EN** トグルボタンを配置
- JSON辞書ファイルによる管理:
  - `src/i18n/ja.json`
  - `src/i18n/en.json`
- 記事コンテンツは Markdown で言語別に管理:
  - `src/content/origins/ethiopia.ja.md`
  - `src/content/origins/ethiopia.en.md`
- URL構造: `/ja/origins/ethiopia` / `/en/origins/ethiopia`
- デフォルト言語: 日本語

## 非機能要件

- **パフォーマンス**: Lighthouse スコア 90+ を目標
- **アクセシビリティ**: セマンティックHTML、適切なalt属性、コントラスト比確保
- **SEO**: メタタグ、OGP、構造化データ対応
- **画像最適化**: Astro の `<Image>` コンポーネントによる自動最適化（WebP/AVIF）
